# Appunti e Progetti

Questa cartella contiene materiale di studio e relazioni di progetti universitari. Di seguito una breve descrizione dei file presenti:

*   **Appunti_automative.pdf**
    Appunti relativi alla guida autonoma e alle architetture software per l'automotive. Vengono trattati argomenti come i livelli di automazione SAE, la sensoristica (Lidar, Radar, Camera), la stima dello stato e gli standard come AUTOSAR (Classic e Adaptive).

*   **Appunti_cognitive.pdf**
    Minimi appunti del corso di Robotica Cognitiva. Il documento copre i concetti fondamentali di ROS (Robot Operating System) e l'intelligenza artificiale conversazionale, con focus sul framework RASA per la creazione di chatbot.

*   **Progetto_APS.pdf**
    Relazione tecnica per un progetto di sicurezza (APS). Descrive un sistema per l'erogazione e la verifica sicura di credenziali digitali, coinvolgendo attori come Utenti (con CIE), Fornitori e Server, con analisi delle minacce e protocolli crittografici.

*   **Progetto_Embedded.pdf**
    Relazione del progetto di Sistemi Embedded. Dettaglia la progettazione e l'implementazione della logica di controllo per un cancello automatico, utilizzando diagrammi di stato (Stateflow) e definendo i requisiti di sistema.
